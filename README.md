My React/Redux Project
---

This is my starter project based on https://github.com/verekia/js-stack-from-scratch tutorial by verekia.com


Install yarn
---
```
npm install -g yarn
```


Start Project
---
```
yarn init -y
```


Add package
---
```
yarn add <package>
```


Remove package
---
```
yarn remove <package>
```
